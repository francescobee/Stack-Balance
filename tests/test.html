Stack & Balance — Test Harness

Smoke tests sulle funzioni pure di js/rules.js. Nessuna libreria. Apri questa pagina in un browser e clicca Run Tests (o vengono eseguiti automaticamente al caricamento).

Aggiungere nuovi test: scrivi describe("Group", () => { it("name", () => assert(...)) }) in tests/test-rules.js (o crea un nuovo tests/test-*.js e aggiungilo allo script-tag list qui sotto). Tutte le globali del gioco (BALANCE, state, computeAwards, …) sono accessibili.
